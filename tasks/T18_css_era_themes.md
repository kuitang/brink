## Description

- [ ] 5 themes implemented:

  - `cold-war`: Typewriter fonts, institutional gray, declassified document

- [ ] Applied via `<body class="theme-{name}">`
- [ ] CSS custom properties for colors, fonts, spacing
- [ ] WCAG AA contrast maintained
- [ ] All game elements styled consistently per theme
